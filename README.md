This project was created as part of my backend re-entry path, focusing on modern Spring Boot and JPA best practices.

# Data Processor API

Spring Boot backend project.

## Features

- CRUD operations
- DTO pattern
- Validation
- Global exception handling
- Pagination and sorting
- Dynamic filtering using JPA Specifications
- Category management
- Tag management
- Many-to-many association entity pattern
- Transaction management

## Tech Stack

- Java 17
- Spring Boot
- Spring Data JPA
- H2 Database
- Maven

## Running

mvn spring-boot:run

## H2 Console

http://localhost:8080/h2-console
